# Audit anti-IA — scene1.ink

**Statistiques brutes :** 111 tirets cadratins (—), 22 points de suspension, 1748 lignes.
**Portée :** tout le texte narratif et les dialogues. Tags inkjs, knots, conditions : intouchés.
**Convention :** pour chaque item, ligne(s) → texte problématique → réécriture proposée → raison (1 phrase).

Les noms "Étoile" du brief = **Stella** dans ce script (nom officiel du projet). Je garde Stella.

## Méta

Les choix de Stella et les répliques de Lunae respectent bien les deux voix :
- **Stella** : phrases cassées, `jsp`, hésitations, auto-interruption. Voix adolescente stressée tenue. ✓
- **Lunae** : phrases complètes, presque trop propres, silences posés. Voix "quelqu'un qui récite" tenue. ✓

Les dialogues ne se terminent presque jamais par le prénom de l'autre. ✓

Le "dark grounded" passe par des détails concrets (highlighter ouvert, cartable, fiches de révision, poubelle, alarme de vélo, dashi dans l'izakaya). ✓ **Pas de** cœurs lacérés, miroirs brisés, silences assourdissants.

**Le problème est à 90% typographique et rythmique.** Le lexique et l'imagerie tiennent. Ce qui fait "IA" dans ce script, c'est le tiret cadratin répété + la construction en phrases nominales équilibrées dans la narration.
